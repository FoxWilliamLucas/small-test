## How To Install

#### First Of All:
- open your terminal 
``` 
git clone https://github.com/FoxWilliamLucas/small-test.git
cd small-test
```
###### windows:
```
wsl --user root
alias sail='[ -f sail ] && bash sail || bash vendor/bin/sail'
sail up -d
```
###### Linux:
```
alias sail='[ -f sail ] && bash sail || bash vendor/bin/sail'
sail up -d
```
###### For testing:
```
sail artisan test
```
